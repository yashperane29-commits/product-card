## ✨ Features & What I Learned

During the development of this project, I tackled several real-world UI design challenges and mastered the following concepts:

* **Advanced Flexbox Layouts:** Learned how to align multi-sized cards seamlessly using `flex-direction: row` and `justify-content: space-evenly`.
* **Dynamic Sizing with Flex-Grow:** Mastered `flex-grow: 1` to ensure that even with different image heights and paragraph lengths, all cards maintain a uniform height and the action buttons stay perfectly aligned at the bottom.
* **Aspect Ratio Management:** Handled non-uniform product images using `object-fit: cover` and fixed aspect-ratios so images look professional without stretching or distortion.
* **Form Element Customization:** Stripped away browser-default styles using `appearance: none` and custom WebKit selectors to build a sleek, modern HTML5 product color picker.
* **Modern Styling & Theming:** Utilized CSS Custom Variables (`--btn`, `--para`) for clean code and implemented an eye-catching `linear-gradient` background.
* **100% Responsive Design:** Implemented CSS Media Queries (`@media`) to shift the desktop row layout into a clean, scrollable vertical column stack on mobile and tablet devices.
